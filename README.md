# Template NPM

## Introduction

This is a template for NPM projects that helps you generate a project supporting both `CommonJS` and `ES Module`
libraries.

## Quick Start

### Download

    git clone git@github.com:lentiancn/template-npm.git
    
    or
    
    git clone git@gitee.com:lentiancn/template-npm.git

### Write code for your library

    src/...
    test/...

### Run test

    npm run test

### Deployment

    npm publish

### Documentation can be provided if needed.

    docs/...
