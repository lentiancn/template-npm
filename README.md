# Template NPM

English | [简体中文](README_zh-Hans.md) | [繁體中文](README_zh-Hant.md)

## Introduction

This is a template for NPM projects that supports both `CommonJS` and `ES Module` target libraries.

## Quick Start

### Download

    git clone git@github.com:lentiancn/template-npm.git
    
    or
    
    git clone git@gitee.com:lentiancn/template-npm.git

### Write code for your library

    src/...
    test/...

### Run tests as expected

    npm run test

### Provide documentation (on demand)

    docs/...

### Deployment

    npm publish
