# Template NPM

[English](README.md) | 简体中文 | [繁體中文](README_zh-Hant.md)

## 简介

这是一个适用于 NPM 工程的模板，可同时支持 `CommonJS` 和 `ES Module` 目标库。

## 快速开始

### 下载

    git clone git@github.com:lentiancn/template-npm.git
    
    或
    
    git clone git@gitee.com:lentiancn/template-npm.git

### 为你的库编写代码

    src/...
    test/...

### 按预期运行测试

    npm run test

### 部署

    npm publish

### 提供文档（按需）

    docs/...
