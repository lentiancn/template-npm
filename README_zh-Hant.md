# Template NPM

[English](README.md) | [简体中文](README_zh-Hans.md) | 繁體中文

## 簡介

這是一個適用於 NPM 工程的模板，可同時支援 `CommonJS` 和 `ES Module` 目標庫。

## 快速開始

### 下載

    git clone git@github.com:lentiancn/template-npm.git
    
    或
    
    git clone git@gitee.com:lentiancn/template-npm.git

### 為你的庫編寫程式碼

    src/...
    test/...

### 按預期執行測試

    npm run test

### 提供文件（按需）

    docs/...

### 部署

    npm publish
